# Changelog

All notable changes to Memstead are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and the project aims to follow [Semantic Versioning](https://semver.org/spec/v2.0.0.html)
once it reaches a tagged release.

## [Unreleased]

Memstead is pre-release: there is no tagged version yet, and the install path is
build-from-source (see [CONTRIBUTING.md](CONTRIBUTING.md)). Changes accumulate
here until the first release is cut, at which point this section is renamed to
that version and a fresh `[Unreleased]` opened above it.

### Added
- Initial public release of the open engine: the schema layer, the in-memory
  store, the folder and git-branch storage backends, the `memstead` CLI, and the
  `memstead-mcp` MCP server.
