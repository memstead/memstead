## Destination: Realizations

- read what already exists before writing — understand the current state and preserve what still works
- the source is the authority. if the realization contradicts the source, the realization is wrong — fix it
- do not rewrite from scratch unless fundamentally broken. prefer targeted changes that bring the realization in line with the source
- respect existing style, patterns, and conventions unless the source explicitly requires a different approach
- when the source is ambiguous about implementation details, make reasonable choices and move on — the source defines what, not how
